# Anna's Birthday Surprise Website

Website sederhana untuk kado ulang tahun Anna Azzahra.

## Tech Stack
- HTML5
- CSS3 (Animations & Layout)
- JavaScript (Vanilla)

## Cara Deploy ke Vercel
1. Upload folder ini ke repository **GitHub** kamu.
2. Login ke **Vercel**.
3. Klik **"New Project"**.
4. Import repository GitHub tersebut.
5. Klik **"Deploy"**. Selesai!
